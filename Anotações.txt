Organização de página:

Header - Topo com título e navegação <header>
Main - Conteúdo principal com a informação central <main>
Sections - Blocos de conteúdo para separar assuntos <section>
Footer - Rodapé com informações finais <footer>
Nav - Define uma área de navegação com links, essencial para a usabilidade e para guiar o usuário pela página <nav>
Aside - Contém conteúdo complementar ou secundário, como barras laterais, anúncios ou informações relacionadas ao conteúdo principal <aside>

Exemplo:

<header>
    <h1>Meu site</h1>
    <nav>
    <a href="#">Início</a>
    <a href="#">Sobre</a>
    <a href="#">Contato</a>
    </nav>
</header>

<main>
    <section>
        <h2>Conteúdo principal</h2>
        <p>Este é o texto central da minha página.</p>
    </section>

    <aside>
        <p>Esta é uma informação complementar.</p>
    </aside>
</main>

<footer>
    <p>Rodapé do site - Direitos reservados.</p>
</footer>

Tags de listas

<ul> e <li>: lista não ordenada com marcadores (unordered list e list item)

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol> e <li>: Lista ordenada com números (ordered list e list item)

<ol>
    <li>Primeiro</li>
    <li>Segundo</li>
    <li>Terceiro</li>
</ol>

Usar ul quando a ordem não importa (hobbies, características). Usar ol quando há sequência (passos de receita, instruções).

Tags gerais:

<h1>: Título principal da página. Use apenas um por página para SEO.
<p>: Parágrafo de texto. Bloco de conteúdo textual.
<img>: Insere imagem. Atributos scr (origem) e alt (descrição).
<a>: Link clicável. Atributo href com destino.

Exemplos:

<h1>Título principal</h1>
<p>Texto do parágrafo</p>
<img src="foto.png" alt="Descrição">
<a href="#">Texto do link</a>

Quando utilizar tabelas:

- Dados comparáveis entre si.
- Informações estruturadas em categorias.
- Relatórios, notas, preços, horários e etc.

Quando não utilizar tabelas:

- Criar layout de página.
- Posicionar elementos visuais.
- Substituir CSS ou Flexbox.

Tabela é para dados, não para design.

Estrutura básica:

<table> Envolve toda a tabela.
<tr> Table Row - define cada linha.
<td> Table Data - define cada célula.
<th> Usado no lugar do td como "título" ou para definir o nome da coluna.

<table>
    <tr>
        <th> Exemplo de Título </th>
    </tr>
    <tr>
        <td> Exemplo de informação da célula</td>
    </tr>
</table>

<lable for="usuario">
    Nome de usuário:
</lable>

<input type="text"
       id="usuario"
       name="usuario"/>

Existem diversos tipos de inputs:

text, email, password, number, radio (marcador que aceita uma única opção, bolinhas), checkbox (marcador que aceita várias opções, quadradinhos), date, file...

Para o text temos: maxlength (máximo de caracteres), minlength(mínimo de caracteres), required(campo obrigatório) e value (define valor padrão, como por exemplo +55 para o DDI do Brasil).

<lable for="estado">Estado:</lable>

<select id="estado" name="estado"> (serve para aqueles botões que monstram um monte de opções pra selecionar)
    <option value="">Selecione...</option>
    <option value="">São Paulo</option>
    <option value="">Rio de Janeiro</option>
    <option value="">Minas Gerais</option>
</select>
